Classic brick breaker game developed as an Android application for the course project. The player controls a paddle to bounce a ball, break bricks, and progress through multiple levels.

## Features

- Paddle and ball movement with collision detection.
- Multiple levels with increasing difficulty.
- Final result screen showing score and game outcome.
- Improved visuals for bricks, paddle, background, and UI.

## Installation

1. Clone the repository:
```git clone https://github.com/EugenioRibon/AndroidProjectBrickBreaker.git```

3. Open the project in **Android Studio**.
4. Let Gradle sync and build the project.

## How to Play

- Move the paddle horizontally to keep the ball from falling.
- Break all bricks to advance to the next level.
- The game ends when you lose all lives or finish all levels, then the final screen shows your result.

## Technologies

- Android Studio
- Kotlin/Java (Android)
- XML layouts and basic game loop logic
